# Contributing to Warlock Toolchain

Warlock is an umbrella project. Most implementation work belongs in the repository for the affected component:

- [Demon Docs](https://github.com/Lokee86/demon-docs)
- [Arcana](https://github.com/Lokee86/arcana-graph)
- Grimoire, once its repository exists
- Pitlord, once its repository exists

This repository owns the shared project description, terminology, toolchain boundaries, integration direction, and roadmap.

Contributions here should:

- distinguish available behavior from active or planned work;
- avoid presenting intended integrations as implemented;
- preserve the deterministic-first and model-independent positioning;
- keep each component independently useful;
- prefer concrete interoperability contracts over broad shared abstractions; and
- link detailed implementation guidance to the owning component repository.

Component repositories retain their own contribution guidance, release process, and license.
